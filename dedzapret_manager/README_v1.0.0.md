**Windows-утилита для управления DPI-desync runtime (zapret/winws2)**

## 🚀 Обзор

DedZapret Manager - это мощный инструмент для управления обходом DPI-блокировок через winws2 runtime. Приложение предоставляет удобный интерфейс для настройки, тестирования и мониторинга стратегий обхода.

## 🔧 Конфигурация

Основной конфигурационный файл находится в:
`%APPDATA%/DedZapretData/config.yaml`

Пример конфигурации:
```yaml
runtime:
  winws2:
    executable: "winws2.exe"
    auto_start: false
    
strategies:
  auto_load: true
  auto_pick_best: true
  
logging:
  level: "INFO"
  rotation: true
  
security:
  mask_sensitive_data: true
```

## 🐛 Устранение неполадок

### Частые проблемы

**Проблема**: winws2 не найден
**Решение**: Убедитесь что winws2.exe находится в `runtime/winws2/bin/`

**Проблема**: Нет прав администратора
**Решение**: Запустите приложение от имени администратора

**Проблема**: Стратегия не работает
**Решение**: Проверьте логи в `logs/` и попробуйте другую стратегию

### Логирование

Логи сохраняются в:
- `%APPDATA%/DedZapretData/logs/app.log` - основной лог
- `%APPDATA%/DedZapretData/logs/runtime.log` - лог runtime
- `%APPDATA%/DedZapretData/logs/session_*.jsonl` - сессии
